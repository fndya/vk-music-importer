Как установить:
1. Открой chrome://extensions/
2. Включи "Режим разработчика"
3. Нажми "Загрузить распакованное расширение"
4. Выбери папку vk-music-importer

Как пользоваться:
1. Открой страницу vk.com/audios...
2. Нажми "Загрузить txt"
3. Выбери txt, где каждая строка — отдельный запрос
4. Расширение начнет с последней строки
5. Добавляй трек вручную в ВК
6. Жми "Следующий" для отметки [done] и перехода дальше
7. Если трек не найден — "Не найдено"
8. "Экспорт" скачает новый txt со статусами

Горячие клавиши:
- Alt + ]  -> Следующий
- Alt + [  -> Назад
- Alt + \ -> Не найдено
